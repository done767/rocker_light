# Abstract

This manual shows how to use Rstudio online for Mac users.

# Manual

download docker for Mac

* https://hub.docker.com/editions/community/docker-ce-desktop-mac

change permission

```
sudo chmod 777 /private/tmp
```

download image and run container

```
chmod +x run.sh
./run.sh
```

save a container

```
chmod +x save.sh
./save.sh
```
